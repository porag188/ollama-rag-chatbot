## How to Contribute

1. **Fork the repository** and clone it:
```bash
git clone https://github.com/OpenRAG-Labs/ollama-rag-chatbot.git
```
2. Create a new branch for your changes:
```bash
git checkout -b feature/your-feature
```
3. Make your changes (code, docs, tests).
4. Commit your changes with a descriptive message:
```bash
git commit -m "Add feature X"
```
5. Push your branch:
```bash
git push origin feature/your-feature
```
6. Open a **Pull Request (PR)** and describe your changes.

## Contribution Rules
- All PRs must pass tests and linting checks before merging.
- PRs should be reviewed by at least one maintainer.
- Keep commits small and descriptive.
- Follow the code style used in the project (PEP8 for Python).
